# Candlestick Strategy

此策略根據使用者提供的《日本蠟燭圖技術新解》筆記整理成可執行規則。核心原則是「位置比型態更重要」，因此系統不會只因為出現單一 K 線就買進，而會同時檢查趨勢、支撐壓力、缺口、差異指數與風險收益比。

## 多頭加分

- 錘子線出現在跌勢或支撐區
- 多頭吞噬、刺穿形態、啟明星出現在趨勢末端
- 上升窗口仍未封閉
- 三線反向突破轉白，收盤突破前三根高點
- 彈簧線：跌破支撐後收回
- 差異指數低於 -10%，代表可能超賣

## 空頭扣分

- 流星線、烏雲蓋頂、空頭吞噬、黃昏星出現在高位或壓力區
- 下降窗口未回補
- 三線反向突破轉黑，收盤跌破前三根低點
- 上插線：突破壓力後收回
- 迭創新高 8 根以上，代表過度延伸
- 差異指數高於 10%，代表價格相對均線過度伸展

## 進出場

- 進場：收盤確認後，或回測支撐不破時分批進場。
- 停損：多頭跌破形態低點或上升窗口封閉；空頭突破空頭形態高點。
- 出場：抵達前高/壓力、出現十字星/連續紡錘線、三線反向轉黑、差異指數背離或風險收益比惡化。
- 資金管理：估算風險收益比低於 2:1 時扣分，避免只為了型態漂亮而交易。
